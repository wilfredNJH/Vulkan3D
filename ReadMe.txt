***KEYS***
'ESC' - close program

'M' - toggle between first person & third person view 

'1' - view skull model 
'2' - view car model 

**THIRD PERSON VIEW**
'MOUSE RIGHT' - pan camera 

**FIRST PERSON VIEW**
'W' - translate forward
'S' - translate backwards
'D' - translate right
'A' - translate left

'UP' - look up 
'DOWN' - look down
'LEFT' - look left
'RIGHT' - look right

'Q' - pitch down
'E' - pitch up

***MOUSE***
Scroll up - translate forward
Scroll down - translate backwards

***MISC***
enableRenderDoc - is automatically enabled whenever in debug mode, can be manually toggled in vk_device.hpp
enableValidationLayers - is automatically enabled whenever in debug mode, can be manually toggled in vk_device.hpp
